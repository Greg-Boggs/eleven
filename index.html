Drupal 8 Admin Theme: Eleven

Eleven theme a Drupal 8 admin theme that makes Drupal not look like drupal!
The admin theme eleven is build for Drupal 8 with the single purpose to make Drupal look prettier and more up to date.
The existing admin theme have been around for at least 7 years, and is beginning to feel a bit dated.
The goal for Eleven theme is to create a beautyfull experience for sitebuilders, content editors, themers, designers and Drupal developers
Download now!

Eleven Features

Icons

An admin theme dont have to be dull + the rest of the world use icons

Moderen UI

All the table listings have been removed for a modern card approach

Colors

Theres more than one tone of Drupal blue in the world

Pretty

Drupal should be as pretty on the outside as the inside

Build on core

Build on Drupal8 core classy theme - No need to install 3rd part modules

Basetheme

Build as a basetheme

Free and open Source

Do with Eleven as you want - turn it up to 12 ... ,)

Support!

Eleven theme is supported by Theme machine (formerly geek royale) that have been in Drupal since 4.7 and know where all the theming stuff is burried in Drupal 8

Modern browsers

Looking forward not back

twig, html, css & js
